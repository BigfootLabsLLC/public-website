# Bigfoot Labs Website

This is the public website for Bigfoot Labs, built with React, Vite, and Tailwind CSS.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Adding Products

1.  Create a new folder in `src/pages/products/`.
2.  Create a component for the product page.
3.  Add a route in `src/App.jsx`.
4.  Add a link card in `src/pages/Home.jsx`.

## Deployment

This site is configured to be deployed to GitHub Pages.
The `dist` folder contains the static assets after running `npm run build`.
